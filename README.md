# 🤖 Java Smart ChatBot

A simple rule-based chatbot built using Java that simulates intelligent conversation using keyword matching and dynamic responses.

## 🚀 Features

* 💬 Interactive chat system
* 🎲 Random responses for natural conversation
* ⏰ Displays current time
* 📅 Displays current date
* 🧠 Keyword-based smart replies
* ❌ Exit command to end chat

## 🛠️ Technologies Used

* Java
* Scanner Class
* Random Class
* Java Time API

## 📌 How It Works

The chatbot reads user input and matches it with predefined keywords. Based on the input, it generates appropriate responses. If no match is found, it returns a fallback message.

## ▶️ How to Run

```bash
javac ChatBot.java
java ChatBot
```

## 💡 Example

You: hi
ChatBot: Hello there!

You: what is time
ChatBot: Current time is 14:32
